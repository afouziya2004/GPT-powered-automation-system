# 🤖 GPT-Based AI Automation System (Capstone Project)

## 📌 Project Overview

This project is developed as part of the **Infosys AI Course Capstone Project** to strengthen practical knowledge gained during the learning program.

It is a **GPT-powered automation system** that performs multiple intelligent tasks using Natural Language Processing (NLP) and Generative AI.

---

## 🎯 Objectives

* Apply Generative AI concepts in real-world scenarios
* Automate common professional tasks using GPT
* Build an end-to-end AI application with modular design
* Enhance practical implementation skills beyond theoretical learning

---

## 🚀 Key Features

### 🔹 1. Automated Data Analysis Code Generation

* Generates Python code for:

  * Exploratory Data Analysis (EDA)
  * Data cleaning and preprocessing
  * Data visualization using Matplotlib & Seaborn

---

### 🔹 2. AI Resume Generator

* Converts a simple paragraph into a **professional resume**
* Structures output into:

  * Skills
  * Experience
  * Projects

---

### 🔹 3. Interview Question Generator

* Generates technical interview questions
* Supports multiple programming languages (Python, Java, etc.)
* Useful for placement preparation

---

### 🔹 4. Meeting Notes Summarizer

* Converts raw meeting notes into **concise summaries**
* Extracts key decisions and updates

---

## 🧠 Technologies Used

* Python
* OpenAI GPT API
* Natural Language Processing (NLP)
* Streamlit (for UI)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone <your-repo-link>
cd gpt-capstone-project
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Set API Key

```
export OPENAI_API_KEY="your_api_key"
```

---

## ▶️ Run Application

### 🔹 Run Streamlit UI

```
streamlit run app/app.py
```

### 🔹 Run CLI Version

```
python main.py
```
